# GloSkin TikTok Script Prompt

Write a TikTok script for GloSkin.

Required markdown sections, exactly in this order:

HOOK
BEATS
CTA
SHOTLIST

Rules:
- HOOK is one short spoken line.
- BEATS is a numbered list.
- CTA points to checking your Glo Score.
- SHOTLIST should use app screenshots, selfies, product labels, and simple text
  overlays.
- Voice is punchy and skincare-native, not clinical.
- Keep claims cosmetic/educational. No cure/treat/prevent language.

Return only the script markdown.
